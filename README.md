# Pulse Grid ⚡

Un juego arcade original con mecánica de onda de energía que recorre una cuadrícula dinámica. Inspirado en los arcades clásicos con estética neon futurista.

## Mecánica

Controlas una **onda de energía** que avanza automáticamente por un grid. Tu objetivo es activar nodos, evitar glitches y sobrevivir el mayor tiempo posible.

- **Cambiar dirección**: Flechas del teclado o toque en móvil
- **Pulso energético**: Barra espaciadora / pulsación larga
- Los nodos activados suman puntos y aumentan el combo
- Los glitches aparecen en zonas inestables y persiguen la onda
- Cada nivel aumenta la velocidad y la densidad de nodos

## Cómo jugar

Abre `index.html` en tu navegador.

O desde el prompt.md se puede ver el diseño conceptual completo del juego.
